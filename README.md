
# Project title
Nykaa marketing campaign performance dashboard

# Project Overview
The project analyzes Marketing campaign performance for nykaa , evaluating campaign effectiveness, customer engagement, channels, ROI and revenue using Excel and Power BI

# key business questions

1) Campaign performance

   - Which campaign type generates highest revenue ?
   - Which campaign type has highest ROI ?
   - Which camapaign type drives most conversions ?

3) Customer segment analysis

 - Which customer segment converts best?
 - Which segment generates highest revenue ?
 - Which segemnt has highest engagement score ?

4) Audience Analysis
 
 - Are working women, college students or youth more responsive ?
 - Which audience generate highest revenue per campaign type ?

- SOCIAL MEDIA (5.76bn Revenue)  AND INFLUENCER (5.75bn Revenue)CAMPAIGN GENERATED HIGHEST REVENUE.
- WORKING WOMEN HAD HIGHER (20.28%) CONVERSIONS .
